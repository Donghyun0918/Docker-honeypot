# HoneyForge

> Docker 기반 멀티유저 허니팟 시뮬레이션 플랫폼.
> 공격 시나리오를 자동 실행하고 수집된 로그를 ML 학습용 데이터셋으로 변환한다.

---

## 프로젝트 개요

| 항목 | 내용 |
| --- | --- |
| 플랫폼명 | HoneyForge |
| 목적 | 사이버 공격 시뮬레이션 + ML 데이터셋 수집 |
| 환경 | 로컬 폐쇄망 (교육·연구 전용) |
| 핵심 기술 | Docker, FastAPI, React, SQLite, Recharts |
| 허니팟 종류 | 8종 (SSH / HTTP / FTP / SMTP / SMB / ICS 등) |
| 공격 시나리오 | 9종 (랜덤화 전술 라이브러리 기반) |

---

## 핵심 기능

### 🍯 허니팟 환경
- 8종 허니팟 컨테이너를 유저별 독립 네트워크로 자동 프로비저닝
- 회원가입 시 컨테이너 세트 자동 생성 / 계정 비활성화 시 자동 정리

### ⚔️ 공격 시나리오
- 9종 시나리오 × 전술 라이브러리 기반 랜덤화
- 실행할 때마다 다른 공격 패턴 생성 (유저명·패스워드·페이로드·타이밍 모두 랜덤)

### 📊 데이터 파이프라인
- 허니팟 로그 파싱 → `dataset.csv` (29컬럼, v4.0 스키마)
- 품질 검증 → ML 전처리 → `dataset_ml.csv`

### 🖥️ 웹 대시보드
- 실시간 로그 스트리밍 (WebSocket)
- ScenarioMonitor: 시나리오 실행 중 허니팟별 활성 상태 시각화
- 통계 차트 (허니팟별·프로토콜별·이벤트 타입·시간대)
- 시나리오 실행 이력 DB 저장
